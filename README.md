## Advanced GPS Location Tracker Tool

HDN-Locator is a powerful Python-based GPS tracking tool that captures exact coordinates of any device. It creates a convincing fake YouTube video page that automatically requests location access. Once the target grants permission, their precise location is captured and displayed on a real-time map.

## ✨ Features

- 🎯 **Exact GPS Coordinates** - Captures latitude, longitude & accuracy
- 🗺️ **Real-time Map** - Built-in Leaflet map visualization
- 🛡️ **Anti-Spoofing** - Detects fake/mock GPS apps
- 🔗 **Public Link** - Cloudflare tunnel for external access
- 📱 **Cross-Platform** - Works on Windows, Linux & macOS
- 🎬 **Fake YouTube Page** - Convincing UI design
- 💾 **Auto Save** - All data saved in JSON format
